ABM Mock Service

Mock service for testing Elastic Synthetics monitoring scripts that validate ABM servers.

Available Endpoints

Frontend IPs API

prbg/api/active-ips/tst/paris3-frontend.json

Returns a JSON array of 3 mock frontend server URLs.

Backend IPs API

prbg/api/active-ips/tst/paris3-backend.json

Returns a JSON array of 3 mock backend server URLs.

Server Health Endpoints

  • servers/frontend1.html
  • servers/frontend2.html
  • servers/frontend3.html
  • servers/backend1.html
  • servers/backend2.html
  • servers/backend3.html

Each returns BalanceCalculationServletProxy - Calc Service Ready

Usage

Base URL: https://pysheriff.github.io/abm-mock-service

The synthetic script should:

  1. Fetch the JSON endpoints to get server URLs
  2. Validate each URL returns HTTP 200 with "Calc" in the response body
